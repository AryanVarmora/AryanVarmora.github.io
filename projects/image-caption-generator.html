Project Overview

The Image Caption Generator is a deep learning-based application that combines Convolutional Neural Networks (CNNs) and Long Short-Term Memory (LSTM) networks to generate descriptive captions for images.

Features

  • Preprocesses and tokenizes textual captions.
  • Extracts image features using pre-trained models like InceptionV3.
  • Generates captions using a CNN-LSTM architecture.
  • Evaluates generated captions using BLEU scores.

Tech Stack

  • Deep Learning: TensorFlow, Keras
  • Computer Vision: OpenCV, InceptionV3
  • Natural Language Processing: NLTK, SpaCy
  • Deployment: Flask, Streamlit (optional)

Dataset

This project utilizes the Kaggle Image Captioning Dataset, which includes images and their corresponding captions.

Challenges Faced

- Training deep learning models on large datasets required extensive computational power.
- Optimizing the BLEU score while maintaining grammatical coherence.
- Handling out-of-vocabulary words effectively in captions.
