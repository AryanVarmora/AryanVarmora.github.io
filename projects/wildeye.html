Overview

WildEye is an AI-driven system for detecting, identifying, and tracking wildlife in their natural habitats. It uses deep learning models and real-time object detection to assist conservation efforts.

Key Features

  • Real-time animal detection and classification.
  • Interactive dashboard for tracking species trends.
  • Data-driven insights for conservation research.

Tech Stack

  • Deep Learning: TensorFlow, PyTorch
  • Computer Vision: OpenCV, YOLOv5
  • Visualization: Dash, Plotly
  • Dataset: Snapshot Serengeti
