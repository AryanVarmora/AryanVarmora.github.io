Project Overview

InsightScribe is an AI-powered tool that transforms raw data into interactive visualizations and meaningful narratives. It enables users to uncover trends, identify outliers, and make data-driven decisions effortlessly.

Key Features

  • Uploads and processes datasets in CSV, Excel, or JSON formats.
  • Automatically generates interactive visualizations with Plotly.
  • Uses AI models (T5, GPT-2) to generate clear, insightful narratives.
  • Allows customization of narrative tone and detail.

Tech Stack

  • Backend: Python (FastAPI, Flask)
  • Data Processing: Pandas, NumPy
  • NLP Models: T5, GPT-2 (Hugging Face Transformers)
  • Visualization: Plotly, D3.js
  • Frontend: React, Vue.js
