Project Overview

GeoInsight is a geospatial analysis tool that processes, analyzes, and visualizes geographical data to uncover spatial patterns and trends.

Key Features

  • Performs advanced geospatial analysis with GIS tools.
  • Creates interactive maps and spatial visualizations using Folium.
  • Supports various geospatial data formats like GeoJSON, Shapefiles, and CSV.

Tech Stack

  • Geospatial Analysis: GeoPandas, Folium, QGIS
  • Data Handling: Pandas, NumPy
  • Visualization: Matplotlib, Leaflet.js
  • Mapping Data: OpenStreetMap, US Census Bureau
