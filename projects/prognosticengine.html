Project Overview

PrognosticEngine is a predictive maintenance system designed to estimate the Remaining Useful Life (RUL) of equipment based on sensor data. By leveraging machine learning techniques and real-time analytics, it helps prevent unexpected failures and optimize operational efficiency.

Key Features

  • Accurately predicts Remaining Useful Life (RUL) using regression and deep learning models.
  • Detects anomalies and abnormal patterns in sensor data.
  • Provides interactive dashboards for machine health monitoring.
  • Modular design to adapt to different industries.

Tech Stack

  • Machine Learning: Scikit-Learn, TensorFlow, PyTorch
  • Data Processing: Pandas, NumPy
  • Visualization: Matplotlib, Plotly, Dash
  • Deployment: Flask, FastAPI

Dataset

The project uses the **CMAPSS dataset** provided by NASA, which simulates degradation data for aircraft engines.
